# Arknights Operators v0.3.2-alpha.1

**Steam Workshop title / Steam 创意工坊标题:** Arknights Operators / 明日方舟干员 [Alpha]

**Automatic localization / 自动本地化:** The selector automatically chooses Chinese or English UI labels from the current game language; operator names prefer Chinese, Japanese, or English when available. / 选择界面会按当前游戏语言自动选择中文或英文界面文字；干员名称会在元数据可用时优先使用中文、日文或英文。

This Alpha release adds multilingual operator discovery and a localized in-game appearance selector for Oxygen Not Included build 740622.

## Highlights

- Search 449 operators by Chinese, English, or Japanese name, PRTS redirect alias, or `char_id`.
- Use automatically selected Chinese or English option labels; Japanese game language prefers Japanese operator display names where PRTS provides them.
- Select an operator, skin, and model through linked controls.
- Select a duplicant and press `Ctrl+F8` to assign its operator, skin, and model; use Mod Options or `Ctrl+Shift+F8` for global defaults.
- Choose a bounded 512 MiB on-demand cache or permanent retention for successfully downloaded appearance resources.
- Render the selected PRTS Spine model and map ONI movement, work, rest, sleep, stress, and death states to available animations.
- Automatically use base models for daily and sleep states, and front combat models for digging, combat, stun, and death.
- Select a duplicant and press `Ctrl+F9` to open its action wheel; choose a visual performance or use the center button to resume ONI state mapping.

Each duplicant can keep an individual operator, skin, and model in the save. Duplicants without an override inherit the global default. First use of an appearance downloads only its small required files from the PRTS resource domain. The package contains code, the lightweight operator catalog, and third-party notices; it contains no Arknights artwork, skeleton, atlas, texture, PRTS web bundle, or local cache.

Start Oxygen Not Included through Steam after installing or enabling the Mod. The locally built and installed release package is `arknights-oni-v0.3.2-alpha.1.zip`. The per-duplicant selector, save reload, action wheel, automatic restoration, original-duplicant hiding, and ground alignment were accepted in a four-duplicant Steam test save.

Earlier local prototypes used the `AmiyaDuplicantMod` directory. The default local installer migrates that directory plus its configuration/cache when the new directory is absent; the hidden legacy identity remains as a compatibility key for existing saves.

# 明日方舟干员 v0.3.2-alpha.1

本 Alpha 版本为 Oxygen Not Included 740622 加入了多语言干员检索和自动本地化的游戏内外观选择界面。

## 主要功能

- 可按中文名、英文名、日文名、PRTS 重定向别名或 `char_id` 搜索 449 名干员。
- 设置界面按游戏语言自动选择中文或英文；游戏语言为日文时，在 PRTS 提供日文名的情况下优先显示日文干员名。
- 干员、皮肤、模型三项联动选择。
- 选中复制人后按 `Ctrl+F8` 单独设置干员、皮肤和模型；从 Mod Options 或 `Ctrl+Shift+F8` 修改全局默认。
- 可选择 512 MiB 上限的按需 LRU 缓存，或永久保留已成功下载的外观资源。
- 直接渲染选中的 PRTS Spine 模型，并将移动、工作、休息、睡眠、压力和死亡状态映射到可用动画。
- 日常与睡眠状态自动使用基建模型，挖矿、战斗、眩晕和死亡自动使用正面战斗模型。
- 选中复制人后按 `Ctrl+F9` 打开动作转盘，可选择视觉表演动作，中心按钮恢复 ONI 状态映射。

每个复制人都可以在存档中保存独立的干员、皮肤和模型；没有单独覆盖的复制人继承全局默认。首次使用某个外观时只会从 PRTS 资源域名下载该外观需要的小文件。发布包仅包含代码、轻量干员目录和第三方声明，不包含明日方舟美术、骨骼、图集、纹理、PRTS 网页 bundle 或本地缓存。

安装或启用 Mod 后请通过 Steam 启动游戏。本地已构建并安装的发布包为 `arknights-oni-v0.3.2-alpha.1.zip`。逐复制人选择、保存重载、动作转盘、恢复自动、原版复制人隐藏和脚底对齐已经在四复制人的 Steam 测试存档中完成验收。

早期本地原型使用 `AmiyaDuplicantMod` 目录。新目录尚未存在时，默认本地安装脚本会迁移旧目录及其配置/缓存；隐藏的旧身份继续作为现有存档的兼容键。
