# Arknights Operators v0.3.3

**Steam Workshop title:** Arknights Operators / 明日方舟干员 [0.3.3]

**Steam Workshop:** https://steamcommunity.com/sharedfiles/filedetails/?id=3765340857

## Highlights

- Adds a paged 96px operator gallery with 20 cards per page, visible-page-only loading, retryable failures, and offline name placeholders.
- Adds in-world skin/model preview and a persistent `Apply to this duplicant` flow.
- Adds per-appearance visual scale keyed by actual `char_id + skin + model`, with a `125%` default and a `75–200%` range.
- Adds a configurable `128–2000 MiB` on-demand cache capacity with a `512 MiB` default, while retaining permanent-download mode.
- Keeps 449 operators in the lightweight metadata snapshot and exposes 420 operators with a base movement model; 29 combat-only characters and 30 non-moving skins are excluded from new selections.
- Fixes thumbnail protocol handling, queue stalls, appearance-load timing, preview/application persistence, Mod Options shutdown cleanup, and critter render-state isolation.

## Compatibility

- Oxygen Not Included build 740622
- Base Game and supported DLC content through `supportedContent: ALL`
- Existing Stable identity: `local.arknights_amiya_duplicant`

## Usage

- `Ctrl+F8`: select and apply an operator, skin, model, and appearance scale for the current duplicant.
- `Ctrl+Shift+F8`: configure global defaults, resource retention, cache capacity, automatic model switching, and default scale.
- `Ctrl+F9`: open the visual action wheel; the centre button restores automatic ONI animation mapping.

Appearance artwork and Spine files remain on-demand runtime resources and are not included in the release ZIP.
